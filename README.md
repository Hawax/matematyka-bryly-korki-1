# Matematyka bez stresu

Interaktywna strona do pracy z matematyką: wizualizacje, zadania, generatory ćwiczeń, zapis wyników oraz wersja do druku / PDF.

## Tematy

- Pola figur płaskich
- Ułamki zwykłe i dziesiętne: porównywanie, skracanie, różne mianowniki, miary, wagi, temperatura, zegar
- Figury płaskie: koło, trapez, romb, równoległobok, obwody i pola
- Siatki figur geometrycznych i bryły: sześcian, prostopadłościan, podstawy ścian / krawędzi / wierzchołków
- Kalendarz i ćwiczenia kalendarzowe

## Funkcje

- tryb pełnoekranowy
- generator zadań
- automatyczny zapis wyników
- wersja do druku / PDF

## Uruchomienie

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
