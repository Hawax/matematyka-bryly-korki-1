# Matematyka bez stresu

Interaktywna strona do pracy z matematyką: wizualizacje, zadania, generatory ćwiczeń, zapis wyników oraz wersja do druku / PDF.

## Tematy

- Pola figur płaskich
- Ułamki dziesiętne: miary, wagi, temperatura, zegar
- Siatki figur geometrycznych i proste pole brył
- Kalendarz i ćwiczenia kalendarzowe

## Funkcje

- tryb pełnoekranowy
- generator zadań
- automatyczny zapis wyników
- wersja do druku / PDF

## Uruchomienie

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
